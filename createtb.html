The CREATE TABLE statement is used to create a new table in a database.

Syntex

CREATE TABLE table_name

column1 datatype,

column2 datatype,

column3 datatype,

....

Example

CREATE TABLE Persons

PersonID int,

LastName varchar(255),

FirstName varchar(255),

Address varchar(255),

City varchar(255)
