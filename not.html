SQL NOT Operator

SQL NOT Operator meaning

The NOT operator is used in combination with other operators to give the opposite result, also called the negative result. In the select statement below we want to return all customers that are NOT from Spain:

syntex

SELECT column1, column2, ... FROM table_name WHERE NOT condition;

example

SELECT * FROM Customers WHERE NOT Country = 'Spain';
